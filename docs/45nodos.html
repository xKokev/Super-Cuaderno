Nodos, Elementos y Selectores 


        
            Para interactuar con HTML nos interesa los nodos de tipo elemento y nodos de tiopo texto 
            
            âŒ GET ELEMENT BY CLASS
            Recoge todos los elementos por tags en este cso li (Ya no se usa) 
            console.log(document.getElementsByTagName("li"));

            âŒ GET ELEMENT BY CLASS NAME 
            Recoge elementos por nombre de clase (Ya no se usa) 
            console.log(document.getElementsByClassName("card"));

            âŒ GET BY NAME
            Recoge elementos por el nombre name (Ya no se usa) 
            console.log(document.getElementsByName("nombre"));

            âœ… GET ELEMENT BY ID [Recomendable usar ]
            Recoge selectores unicos  (mas rapido)
            console.log(document.getElementById("menu"));

            âœ… QUERY SELECTOR [Recomendable usar ]
            Consulta de selector y recibe un parametro valido de css ID, CLASE [el primer selector que le hayas indicado es el que trae](es mas lento que el getElementById) 
            console.log(document.querySelector("#menu"));

            âœ… QUERY SELECTOR [Recomendable usar ]
            Sirve para buscar todos los enlaces de una pagina  [pero solo trae al primer selector]
            console.log(document.querySelector("a"));

            âœ… QUERY SELECTOR ALL [Recomendable usar ]
            Me permite traer todo el selector y no solo el primero 
            console.log(document.querySelectorAll("a"));
