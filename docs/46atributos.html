            üêô Como interactuar con los atributos de las etiquetas HTML 
            
            üê≥ Podremos obtener los datos de dos maneras (muchas veces da lo mismo)
            üëÜ Con la NOTACION DEL PUNTO 
            ‚úåÔ∏è Con GET ATTRIBUTE (forma corecta)
            console.log(document.documentElement.lang);

            ü¶ë Otras veces nos da valores distintos 
            console.log(document.querySelector(".link-dom").href);

            console.log(document.querySelector(".link-dom").getAttribute("href"));

            üê° Establecer nuevo valor a los atributos 
            Con notacion del punto 
            document.documentElement.lang = "en";
            console.log(document.documentElement.lang);
            // en

            ü¶û Con SET [nombre del atributo , nuevo valor que queremos]
            document.documentElement.setAttribute("lang", "es-MX");
            console.log(document.documentElement.lang);
            // es-MX 

            üê† Se puede usar let o const hay programadores que le ponemos un $variables a variables que guardan elementos del  DOM

            üêü Modificando elementos del DOM en este caso haremos que nuestro link abra en una nueva pesta√±a 
            const $linkDOM = document.querySelector(".link-dom");
            $linkDOM.setAttribute("target", "_blank");
            
            üê¨ Por seguridad hacer que no haya una dependencia entre la pesta√±a que estamos abriebndo y la ventana origen 
            $linkDOM.setAttribute("rel", "noopener");

            üê¢ Tambien podemos hacer que abra otro link 
            $linkDOM.setAttribute("href", "https://youtube.com/jonmircha");
            
            ü¶à Validando si tenemos el atributo rel y en este caso si porque lo a√±adimos antes 
            console.log($linkDOM.hasAttribute("rel"));
            // true

            üêå Eliminar atributos en el DOM 
            $linkDOM.removeAttribute("rel");
            console.log($linkDOM.hasAttribute("rel"));
            // false 

            üêä DATA ATTRIBUTES  
            DATA ATRIBUTES nos permite crear nuestros propios atributos [pare crear un nuevo data atributes usaremos DATA y un guion data-descripcion por ejemplo ]

            console.log($linkDOM.getAttribute("data-description"));
            // Document Object Modelo
            console.log($linkDOM.dataset);
            // DOMStringMap {description: Document Object Modelo"}

            console.log($linkDOM.dataset.description);
            // Document Object Modelo
            ü¶ì Estableciendo nuevos valores con SET ATTRIBUTES (Nombre del atributo, nuevo valor que deseamos)
            $linkDOM.setAttribute("data-description", "Modelo de Objeto del Documento");
            console.log($linkDOM.dataset.description);
            // Modelo de Objeto del Documento

            ü¶õ Estableciendo nuevos valores CON NOTACION DEL PUNTO (.dataset.descripcion )[recordar que lo habiamos linkeado a youtube]
            $linkDOM.dataset.description = "Suscr√≠bete a mi canal y comparte";
            console.log($linkDOM.dataset.description);
            //  Suscr√≠bete a mi canal y comparte
            
            ü¶è Consulta si tiene algun atributo

            ü¶í Eliminacion de atributos 
            $linkDOM.removeAttribute("data-id");
